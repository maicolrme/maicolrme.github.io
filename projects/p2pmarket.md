# P2P Market

**URL:** [http://p2pmarket.devsee.pro](http://p2pmarket.devsee.pro)

## Description
Local P2P marketplace for cryptocurrency trading. Connects buyers and sellers directly with escrow-protected transactions, supporting Bitcoin, Ethereum, Litecoin, and more.

## Tech Stack
- **Frontend:** Vue.js, Laravel Blade, Bootstrap
- **Backend:** Laravel (microservices architecture)
- **Database:** MySQL, MongoDB, Redis
- **Cloud:** DigitalOcean
- **Containerization:** Docker
- **Real-time:** WebSockets, Event-driven
- **Blockchain:** Bitcoin, Ethereum, Litecoin integrations

## Key Features
- Local P2P trading with geolocation
- Multi-cryptocurrency support (BTC, ETH, LTC, DOGE, CELO, cUSD)
- Escrow-protected transactions
- Real-time chat and negotiation
- Price tracking and market data
- Referral and rewards program
- Multi-language support (EN, ES)
- Advanced search and filtering
- Mobile-responsive design
