# Exchanger21

**URL:** [http://exchanger21.devsee.pro](http://exchanger21.devsee.pro)

## Description
Cryptocurrency exchange platform featuring a high-performance matching engine, P2P marketplace, and real-time trading events. Built for secure and fast digital asset trading.

## Tech Stack
- **Frontend:** Nuxt 4, Vue.js, Tailwind CSS
- **Backend:** AdonisJS (Node.js)
- **Database:** MySQL, Redis
- **Cloud:** DigitalOcean
- **Containerization:** Docker
- **Real-time:** WebSockets, Event-driven architecture

## Key Features
- Order matching engine
- P2P marketplace for direct trades
- Real-time price and order book updates
- Multi-cryptocurrency support
- Wallet integration and management
- Trading history and analytics
- Secure transaction processing
- Docker containerized deployment
