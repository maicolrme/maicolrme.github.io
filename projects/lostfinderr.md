# LostFinderr

**URL:** [https://lostfinderr.com](https://lostfinderr.com)

## Description
Platform focused on asset recovery and lost item tracking. Helps users report, search, and recover lost belongings through a centralized system.

## Tech Stack
- **Frontend:** Vue.js, Nuxt, Tailwind CSS
- **Backend:** Laravel
- **Database:** MySQL, Redis
- **Cloud:** DigitalOcean
- **Real-time:** WebSockets for live updates

## Key Features
- Item reporting and classification system
- Search and matching algorithm for lost items
- User dashboard with claim management
- Notification system for match alerts
- Admin panel for moderation
- Geospatial search by location
