# DevSee

**URL:** [https://devsee.pro](https://devsee.pro)

## Description
Talent recruitment and management platform designed for scaling teams. Connects companies with skilled developers and streamlines the hiring workflow.

## Tech Stack
- **Frontend:** Vue.js, Nuxt, Bootstrap
- **Backend:** Laravel
- **Database:** MySQL, Redis
- **Cloud:** DigitalOcean
- **Authentication:** API Keys, 2FA

## Key Features
- Developer profile management
- Job posting and search
- Application tracking system
- Team management dashboard
- Skill assessment tools
- Communication platform

## Status
Production
