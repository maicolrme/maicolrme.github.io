# Spring Boot + Nuxt 4 CRUD

**URL:** [https://github.com/maicolrme/sprintBootNuxt](https://github.com/maicolrme/sprintBootNuxt)

## Description
Full-stack CRUD application featuring a Spring Boot 3.2 backend and Nuxt 4 frontend. Demonstrates modern Java + Vue.js integration with enterprise-grade authentication and API documentation.

## Tech Stack
- **Frontend:** Nuxt 4, Vue.js, Bootstrap
- **Backend:** Spring Boot 3.2, Java 17
- **Database:** PostgreSQL (with Flyway migrations)
- **Auth:** JWT (stateless), RBAC
- **Docs:** Swagger / OpenAPI
- **Build:** Maven

## Key Features
- JWT stateless authentication
- Role-Based Access Control (RBAC)
- Database migrations with Flyway
- Complete CRUD operations
- API documentation via Swagger/OpenAPI
- Responsive Bootstrap UI
- Docker support
