# Team4Labs

**URL:** [https://team4labs.com](https://team4labs.com)

## Description
Collaborative workspace platform for productive teams. Integrates Kanban boards, shared documents, file management, and team communication in a single workspace.

## Tech Stack
- **Frontend:** Nuxt 4, Vue.js, Tailwind CSS
- **Backend:** Laravel
- **Database:** PostgreSQL, Redis
- **Cloud:** DigitalOcean
- **Real-time:** WebSockets, Server-Sent Events
- **Auth:** JWT, 2FA

## Key Features
- Kanban boards with drag & drop
- Real-time collaborative document editing
- File management with version control
- Team workspace with granular permissions
- Activity feed and notifications
- Calendar and event scheduling
- Diagram and flow chart editor
- Multi-language support (ES, EN, IT)
