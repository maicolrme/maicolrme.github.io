# NettChain PHP SDK

**URL:** [https://github.com/nettchain/phpsdk](https://github.com/nettchain/phpsdk)

## Description
Official PHP SDK for integrating with the NettChain blockchain API. Supports wallet operations, transactions, and smart contract interactions from PHP applications.

## Tech Stack
- **Language:** PHP
- **Dependencies:** Guzzle HTTP, PHP 8.0+
- **API:** RESTful NettChain API

## Key Features
- Wallet creation and management
- Transaction signing and broadcasting
- Smart contract interaction
- Balance and history queries
- Webhook integration support
- PSR-compliant HTTP client
- Comprehensive test coverage

## Installation
```bash
composer require nettchain/phpsdk
```

## Status
Production / Open Source
