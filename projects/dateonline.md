# DateOnline

**URL:** [https://date-online.app](https://date-online.app)

## Description
Premium classifieds and dating/meeting application. Features verified listings worldwide for escorts, companions, and adult services organized by category and city.

## Tech Stack
- **Frontend:** Laravel Blade, Tailwind CSS, Alpine.js
- **Backend:** Laravel 11
- **Database:** MySQL, Redis
- **Cloud:** DigitalOcean
- **Auth:** Laravel Breeze / Sanctum

## Key Features
- Category-based browsing (Escorts, Trans, Gays, Massage, etc.)
- City and country search with filtering
- User registration and profile management
- Ad posting and management system
- Multi-language support (ES, EN, IT)
- Responsive mobile-first design
- OG image and social meta tags
- SEO-optimized structure
